World Of Boats

World Of Boats is also a 2D Game made with Python which I started developping in December 2021 and was finished in February 2022.
This game is a board game inspired by 2 Games: World of Warship and Battleship.


Firstly, It was only a friend who created a command-in-line game with basic things when we learned Object-Oriented coding at school.
Then, we worked together to create a command-in-line game with much more possibilites.
After that, I created all the User Interface using Tkinter module.


This project is the first "serious" and personnal project and I think I created something great even if the code isn't clean and there is obviously a lot of coding mistake.
It was a really great experience and this is with this project that I fell in love with coding, finding solutions to issues, building a Project on your own...


To describe more precisely the game, when you first launch it, you land on a menu where you can add players (up to 3 players on the same team, there is only 2 teams) and select the boat class of the player.
Here is the 4 boat class you can choose (some of the classes name are in french):
-Cuirass√©: A Tank with a lot of Health and Resistance, big Range Attack, not a lot of movement and his type of attack are the shells.
-Cruiser: All his stats (Health / Resistance / Range Attack / Movement / Attack Damage) are equilibrated, and his type of attack are both shells and torpedo.
-Destroyer: A lot of Attack Damage and Movement but not a lot of Health and Resistance. His type of attack are both shells and torpedo.
-Submarine: A lot of Attack Damage / Movement / Range Attack, a lot of Resistance to shells (he is a submarine...) but not a lot of Health. His attack type are the torpedos.

After everyone is added to the game, you can start the game and the next step is to choose your bonus. Every class has his own bonus which can be more movement, health, range, attack damages, resistance and many other stats...

After everyone choosed, the game has started and you can play on a 20x20 square map. Every player of the red team spawn randomly on the left side of the map, and every player of the green team spawn randomly on the right side.
The yellow circle is the range attack of the player which playing his turn.
When a player play his turn, he has 4 possibilites (only 1 can be selected): skip his turn, attack someone, heal himself (only some classes have only few heals) and move.

QR Code Generator

QR Code Generator is a project made in August 2022 that only took me a day to build.
The goal of this project was only to start coding again before my 2022/2023 school year because I did not code for many months.
This app is made with Python and Tkinter module.
Even tho it is a very small and easy project, it was still a great project to do and especially in one day.


To describe how I did this project, I firstly created a function to transform link to QR Code using the qrcode module.
Then, I also created 2 more function: one to copy the QR Code into your clipboard, and the other one to save it.


After I did all the functions, I started working on the User Interface using Tkinter module.
I only had few things to add: a Title, an Input to give the link, a Button to create the QR Code which will show the QR Code after the button is pressed, and 2 more buttons when there is a QR Code to copy into your clipboard and to save it.


With this project, I learned something very important: I made progress with coding.
I realized that when I had this project idea, I already knew all the steps I've just told you, I already knew which functions I had to make, how the User Interface will look like...
And this is completly different from my first project World Of Boats because when I started it, I didn't knew where to start, how to make it...
